# PIANO

PIANO, piyano/müzik temalı 2D Unity oyununun Windows build dosyalarını içerir.
Oyunu hazır build olarak indirip `PİANO.exe` dosyası üzerinden çalıştırabilirsiniz.

## Çalıştırma

1. Repoyu bilgisayarınıza indirin veya klonlayın.
2. Klasör içindeki `PİANO.exe` dosyasını açın.

## Not

Bu repo Unity kaynak projesinden çok, oyunun çalıştırılabilir Windows build çıktısını içerir. `PİANO_Data`, `UnityPlayer.dll` ve `MonoBleedingEdge` klasörleri oyunun çalışması için gereklidir.
